# 📌 Information
### ✨ Introduce
<strong>빨래말래</strong>는 빨래 점수를 알려주는 웹 서비스입니다. <br>
사용자가 선택한 위치의 3일치 데이터(날씨, 습도, 풍속, 강우량, 적설량 등)를 취합하여 빨래 점수를 판단합니다. <br>
기상청에서 제공해 주는 오픈 API를 기반으로 개발되었습니다. 
<br><br>
<strong>ppllae-mallae</strong> is a web service that tells you your laundry score. <br>
The laundry score is determined by collecting three days' worth of data (weather, humidity, wind speed, rainfall, snowfall, etc.) of the location selected by the user. <br>
It was developed based on an open API provided by the Korea Meteorological Administration.

<br><br><br>
